Introduction

Ollin est un langage de script dynamiquement typé compilé vers un bytecode 32-bit et exécuté par une VM register-based. La syntaxe est inspirée de Lua et Python : blocs délimités par end, newline comme terminateur d'instruction, typage implicite.

Pipeline d'exécution

source .ol  →  Lexer  →  Parser  →  Compiler  →  VM

Premier programme

print("hello world!")

var x = 42
print("x = ", x)   ## x = 42

Types

Ollin utilise une struct taguée : 1 octet de tag + union 8 octets = 16 octets par valeur. Ceci permet des entiers 64-bit complets.

TypeLittéralDescription
nilnilValeur absente. Toute variable non initialisée vaut nil.
booltrue, falseStockés comme entiers 1 et 0.
integer42, 1_000Entier signé 64-bit (±263 ≈ ±9.2×1018).
float3.14, .5Double IEEE 754.
string"texte"UTF-8, immuable, alloué sur le tas.
map{}, {"a": 1}Dictionnaire clé→valeur. Toute valeur peut être clé (littéral : string ou ident).
array[], [1, 2, 3]Tableau dynamique, indexé à 1, poolé.

Variables

Toute variable doit être déclarée avant utilisation — utiliser ou affecter un nom non déclaré est une erreur de compilation. var crée une variable locale (registre de la VM) ; global crée une variable globale visible dans tout le programme.

var a = 10
var b = 2.5
var s = "bonjour"
var undef        ## pas d'initialisation → nil
assert(undef == nil)

Déclaration multiple

var x, y = 1, 2
var p, q, r = 10, 20   ## r → nil (pas assez de valeurs)

Affectation

var a = 10
a = 42           ## simple
a += 1           ## compound : += -= *= /= %=
a *= 2
assert(a == 86)
Les underscores dans les littéraux numériques sont ignorés : 1_000_000 == 1000000.

Variables globales — global

Le mot-clé global déclare une variable globale, visible dans tout le programme. Elle peut être déclarée n'importe où et lue ou modifiée depuis n'importe quelle fonction. Une variable locale (var) de même nom masque le global dans sa portée.

global compteur = 0

func incr()
    compteur += 1      ## modifie le global depuis une fonction
end

incr()
incr()
assert(compteur == 2)
print(compteur)        ## 2
var crée toujours une variable locale ; global crée une variable globale (table partagée de la VM). Affecter un nom non déclaré reste une erreur de compilation.

Constantes — const

Le mot-clé const déclare une locale immuable. L'initialisation est obligatoire. Toute tentative d'assignation ultérieure est une erreur de compilation.

const PI = 3.14159
const MAX = 100

assert(PI == 3.14159)
assert(MAX == 100)

func aire(r)
    const DEUX_PI = 2 * PI
    return DEUX_PI * r * r
end

print(aire(1))     ## 6.28318…
Les erreurs suivantes sont détectées à la compilation : 
const x      ## ERROR: must be initialized
const x = 1
x = 2         ## ERROR: cannot assign to const 'x'

Opérateurs bits

Ollin supporte les opérateurs bit-à-bit style C. Ils opèrent uniquement sur des entiers (int48).

Binaires

OpDescriptionExemple
&ET bit-à-bit12 & 10 == 8
|OU bit-à-bit12 | 10 == 14
^XOR bit-à-bit12 ^ 10 == 6
<<Décalage gauche1 << 3 == 8
>>Décalage droit (arithmétique)16 >> 2 == 4

Unaire

OpDescriptionExemple
~Complément à 1~0 == -1, ~1 == -2

Précédence

De la plus basse à la plus haute :

or  →  and  →  |  →  ^  →  &  →  comparaisons  →  << >>  →  + -  →  * / %  →  ~ - not

Exemples

var flags = 0
flags = flags | 1   ## activer bit 0
flags = flags | 4   ## activer bit 2
assert(flags == 5)   ## 0101

assert((flags & 1) == 1)  ## tester bit 0
assert((flags & 2) == 0)  ## bit 1 absent

flags = flags ^ 1   ## basculer bit 0 → 4
assert(flags == 4)

assert(1 << 8 == 256)
assert(256 >> 4 == 16)
assert(~0 == -1)
assert(255 & 15 == 15)
Les opérateurs bit-à-bit nécessitent des opérandes entiers (integer). Appliquer &, |, ^, ~, << ou >> à un float lèvera une erreur à l'exécution.

Chaînes

var s = "bonjour"
assert(s == "bonjour")
assert("" == false)          ## chaîne vide est falsy
assert("texte" == true)    ## non vide est truthy

La concaténation se fait avec + :

var greeting = "Hello, " + "world" + "!"
assert(greeting == "Hello, world!")
Les chaînes sont immuables. Seul + est surchargé pour les strings — les autres opérateurs (-, *…) restent réservés aux nombres.

Expressions multi-lignes

À l'intérieur de parenthèses, les newlines sont ignorés. Cela permet de découper les longues expressions et conditions.

var total = (
    prix_ht
    * ((1 + tva) / 100)
    + frais_port
)

if (commande_valide
    and stock_disponible
    and paiement_ok) then
    expédier()
end
Hors parenthèses, le newline termine l'instruction. then sert de délimiteur explicite pour if — les parenthèses ne sont donc pas obligatoires pour les conditions courtes.

If / Else

if condition then
    ## bloc then
else if autre_condition then
    ## else if (autant que nécessaire)
else
    ## else optionnel
end
var score = 75
var grade = "F"

if score >= 90 then
    grade = "A"
else if score >= 80 then
    grade = "B"
else if score >= 70 then
    grade = "C"
else
    grade = "F"
end

assert(grade == "C")

While

var i = 0
while i < 10
    i += 1
end
assert(i == 10)

## break quitte la boucle immédiatement
var k = 0
while true
    k += 1
    if k >= 5 then break end
end
assert(k == 5)

For

Trois variantes pour les boucles numériques, plus l'itération sur les maps et les arrays.

Range [a;b]

Notation d'intervalles mathématiques — bornes inclusives ou exclusives, step optionnel.

var sum = 0
for i in [1;10]
    sum += i
end
assert(sum == 55)   ## 1+2+…+10 (inclusif des deux côtés)

## droit exclusif : 1,2,...,9
for i in [1;10[
    print(i)
end

## avec step : 1,3,5,7,9
for i in [1;10;2]
    print(i)
end

## first-class : stockable dans une variable
var r = [1;5]
for i in r
    print(i)   ## 1 2 3 4 5
end

Numérique i=start,end[,step]

Step optionnel — positif ou négatif.

## step implicite = 1
for i=1,5
    print(i)   ## 1 2 3 4 5
end

## step positif (pairs)
for i=0,10,2
    print(i)   ## 0 2 4 6 8 10
end

## step négatif (compte à rebours)
for i=5,1,-1
    print(i)   ## 5 4 3 2 1
end

Break dans for

for i in [1;100]
    if i > 3 then break end
    print(i)   ## 1 2 3
end

Fonctions

func add(a, b)
    return a + b
end

assert(add(3, 4) == 7)

Retours multiples

func minmax(a, b)
    if a < b then
        return a, b
    end
    return b, a
end

var lo, hi = minmax(7, 3)
assert(lo == 3 and hi == 7)

Récursion

func fact(n)
    if n < 2 then return 1 end
    return n * fact(n - 1)
end

assert(fact(10) == 3628800)

Arguments variadiques

... capture tous les arguments supplémentaires.

func log(niveau, ...)
    printf("[{}] ", niveau)
    print(...)
end

log("INFO", "démarrage", "ok")  ## [INFO] démarrage ok

Passer les varargs à une autre fonction

func wrapper(a, ...)
    return a, ...   ## propage les varargs
end

var x, y, z = wrapper(1, 2, 3)
assert(x == 1 and y == 2 and z == 3)

Paramètres par défaut

Les valeurs par défaut doivent être des constantes littérales.

func greet(name, greeting="Bonjour")
    printf("{}, {} !", greeting, name)
end

greet("Alice")             ## Bonjour, Alice !
greet("Bob", "Salut")    ## Salut, Bob !
func connect(host="localhost", port=8080, debug=false)
    printf("{}:{} debug={}", host, port, debug)
end

connect()                       ## localhost:8080 debug=0
connect("prod.example.com")   ## prod.example.com:8080 debug=0
connect("prod", 443)         ## prod:443 debug=0

Closures

Une fonction peut capturer les variables de la portée qui l'entoure (upvalues). La valeur capturée reste vivante même après le retour de la fonction externe.

Compteur encapsulé

func make_counter()
    var n = 0
    func inc()
        n += 1
        return n
    end
    return inc
end

var c1 = make_counter()
var c2 = make_counter()   ## état indépendant

assert(c1() == 1)
assert(c1() == 2)
assert(c2() == 1)   ## c2 repart de 0

Fabrique de fonctions

func make_adder(x)
    func add(y)
        return x + y
    end
    return add
end

var add5  = make_adder(5)
var add10 = make_adder(10)

assert(add5(3)  == 8)
assert(add10(3) == 13)

Fonctions imbriquées

Une fonction déclarée à l'intérieur d'une autre reste locale — elle n'est pas ajoutée à l'environnement global.

func process(data)
    func helper(x)   ## visible uniquement dans process
        return x * 2
    end
    return helper(data) + helper(data + 1)
end

assert(process(3) == 14)   ## 3*2 + 4*2

Upvalue partagé entre fonctions

func make_shared()
    var count = 0
    func inc()  count += 1  end
    func get()  return count  end
    return inc, get
end

var inc, get = make_shared()
inc()
inc()
inc()
assert(get() == 3)
print(get())   ## 3

Maps

Dictionnaire clé→valeur. La syntaxe de littéral accepte des chaînes ou des identifiants ; à l'exécution, toute valeur peut servir de clé (nil, entier, float, string, bool, array, map, fonction…).

Création

var vide = {}

var point = {"x": 10, "y": 20}

## clés sans guillemets
var config = {
    host:  "localhost",
    port:  8080,
    debug: false,
}

Lecture

Crochets
point["x"]   ## 10
point["z"]   ## nil
Point (équivalent)
point.x     ## 10
point.z     ## nil

Écriture

var point = {"x": 10, "y": 20}
point["z"] = 0    ## nouvelle clé
point.z = 5      ## équivalent

point["x"] += 1  ## compound via crochet
point.y *= 2     ## compound via point

assert(point.x == 11)
assert(point.y == 40)

Maps imbriquées

var scene = {
    camera: {fov: 60, near: 0.1, far: 1000},
    light:  {x: 0, y: 10, z: 0, intensity: 1},
}

assert(scene["camera"]["fov"] == 60)

Clés de tout type (via crochets)

var m = {}

m[nil]   = "nil key"
m[42]    = "int key"
m[3.14]  = "float key"
m[true]  = "bool key"

## INTEGER(1) et FLOAT(1.0) sont la même clé
m[1] = "one"
assert(m[1.0] == "one")

Sémantique de copie

Les maps sont partagées par référence (comptage de référence). Affecter une map à une autre variable ne la clone pas.
var a = {x: 1}
var b = a      ## b pointe vers la même map
b.x = 99
assert(a.x == 99)  ## a est modifié aussi

Itération sur une map

La syntaxe for k, v in m parcourt toutes les entrées dans l'ordre d'insertion.

var scores = {Alice: 95, Bob: 82, Carol: 78}

for nom, score in scores
    printf("{} → {}", nom, score)
end
## Alice → 95
## Bob → 82
## Carol → 78

Somme des valeurs

var scores = {Alice: 95, Bob: 82, Carol: 78}
var total = 0
for k, v in scores
    total += v
end
printf("moyenne : {}", total / 3)

Dans une fonction

func sum_values(m)
    var s = 0
    for k, v in m
        s += v
    end
    return s
end

var scores = {Alice: 95, Bob: 82, Carol: 78}
assert(sum_values(scores) == 255)

Arrays

Tableau dynamique distinct des maps. Les indices commencent à 1. Les arrays sont poolés et ref-comptés comme les maps.

Création

var vide = []

var t = [10, 20, 30]

var calc = [1+1, 2*3, 10-1]   ## expressions autorisées

Lecture / Écriture

print(t[1])      ## 10  (indexé à 1)
print(t[4])      ## nil (hors bounds → nil)

t[2] = 99        ## set
t[2] += 1        ## compound : += -= *= /= %=

var a = []
a[5] = "x"      ## grossit automatiquement (indices 1..4 → nil)

Itération — valeurs seules

var sum = 0
for v in [1, 2, 3, 4, 5]
    sum += v
end
assert(sum == 15)

Itération — index + valeur

var t = ["a", "b", "c"]
for i, v in t
    printf("{}: {}", i, v)
end
## 1: a
## 2: b
## 3: c

Sémantique de copie

Les arrays sont partagés par référence (comptage de référence), comme les maps.

Gestion d'erreurs

Ollin utilise try / catch / else / end. Le bloc else est exécuté si aucune exception n'est levée. Tous les blocs sont optionnels.

try
    throw "quelque chose s'est mal passé"
catch err
    print("erreur :", err)
else
    print("tout s'est bien passé")
end

Throw de n'importe quel type

func diviser(a, b)
    if b == 0 then
        throw {code: 42, msg: "division par zéro"}
    end
    return a / b
end

try
    var r = diviser(10, 0)
catch e
    printf("erreur {}: {}", e["code"], e["msg"])
end

Try sans corps

try
catch err
    print(err)
end

Fonctions intégrées

print

Affiche tous les arguments séparés par des espaces, suivi d'un retour à la ligne.

print("a =", 42, true)   ## a = 42 1
print()                    ## ligne vide

printf

Formatage avec {} pour substitution positionnelle ou {n} pour indexation explicite.

printf("{} + {} = {}", 1, 2, 3)         ## 1 + 2 = 3
printf("{0} et {0}", "oui")             ## oui et oui
printf("x={0} y={1} x={0}", 10, 20)   ## index explicite

assert

Lève une exception si la condition est falsy. Message optionnel.

var x = 42
assert(1 + 1 == 2)
assert(x > 0, "x doit être positif")

time

Retourne le temps UNIX en secondes (float). Utile pour les benchmarks.

var t0 = time()

## ... code à mesurer ...

var t1 = time()
printf("durée : {} s", t1 - t0)

Exemple complet

## Benchmark : somme de 1 million d'itérations
var N = 1_000_000
var t0 = time()

var s = 0
for i in [1;N]
    s += i
end

var t1 = time()
printf("somme = {}  temps = {} s", s, t1 - t0)

Fonctions intégrées comme valeurs

print, printf, assert et time sont des valeurs de première classe — assignables, passables en argument :

var f = print
f("hello")               ## hello

func apply(fn, x)
    fn(x)
end
apply(print, 42)          ## 42

Import

L'instruction import charge un fichier Ollin et injecte son contenu dans le programme courant au moment de la compilation (résolution statique, pas de sous-VM).

Les chemins sont relatifs au répertoire du script courant. L'extension .ol est ajoutée automatiquement si absente.

Import plat

Toutes les définitions du fichier importé deviennent disponibles directement dans le scope courant.

## utils.ol
func add(a, b)
    return a + b
end
var PI = 3.14159

## main.ol
import "utils"
print(add(2, 3))    ## 5
print(PI)           ## 3.14159

Import modulaire

Avec as name, toutes les définitions sont regroupées dans une map nommée.

import "utils" as math
print(math.add(10, 20))   ## 30
print(math.PI)             ## 3.14159

Règles

  • Chemins résolus relativement au répertoire du script courant (récursivement pour les imports imbriqués).
  • Les imports circulaires sont détectés et ignorés (cache partagé).
  • Un même fichier importé deux fois n'est exécuté qu'une fois.

Classes

Ollin supporte l'orienté-objet avec héritage simple, constructeur implicite et méta-méthodes.

Déclaration

class Animal
    func init(name, sound)
        self.name = name
        self.sound = sound
    end
    func speak()
        return self.name + " says " + self.sound
    end
    func __str()
        return "Animal(" + self.name + ")"
    end
end
  • func init(...) est le constructeur ; self est passé implicitement en premier paramètre.
  • Toutes les méthodes reçoivent self automatiquement.

Instanciation

class Animal
    func init(name, sound)
        self.name = name
        self.sound = sound
    end
    func speak()
        return self.name + " says " + self.sound
    end
end
var a = Animal("Dog", "woof")
print(a.speak())   ## Dog says woof
print(a.name)      ## Dog

Héritage

class Animal
    func init(name, sound)
        self.name = name
        self.sound = sound
    end
    func speak()
        return self.name + " says " + self.sound
    end
end
class Dog extends Animal
    func init(name)
        super.init(name, "woof")   ## appel du constructeur parent
    end
    func fetch()
        return self.name + " fetches!"
    end
end

var d = Dog("Rex")
print(d.speak())   ## Rex says woof  (méthode héritée)
print(d.fetch())   ## Rex fetches!

Si un enfant n'a pas de func init, il hérite automatiquement de celui du parent.

super

super.method(args) appelle la méthode du parent avec le self courant.

Méta-méthodes

    func init(x, y)
        self.x = x
        self.y = y
    end

Graphics

Le module graphics expose Raylib pour le rendu 2D en temps réel. Il est disponible automatiquement sans import. graphics.run(callback) fonctionne sur desktop et WASM.

Cycle de vie

graphics.canvas(800, 600, "Titre")   ## ouvre la fenêtre

func frame()
    graphics.begin_draw()
    graphics.clear(graphics.BLACK)

    ## dessin ici

    graphics.end_draw()
end

graphics.run(frame)   ## boucle principale (compatible WASM)

API

FonctionDescription
graphics.canvas(w, h, titre)Ouvre une fenêtre w×h avec le titre donné, 60 fps
graphics.run(callback)Lance la boucle principale ; appelle callback() chaque frame jusqu'à fermeture
graphics.is_open()Retourne 1 si la fenêtre est encore ouverte, 0 sinon
graphics.begin_draw()Commence un frame (appel obligatoire avant tout dessin)
graphics.end_draw()Présente le frame à l'écran
graphics.clear(color)Efface l'écran avec la couleur donnée
graphics.line(x1,y1,x2,y2 [,color])Trace un segment de (x1,y1) à (x2,y2)
graphics.fps()Retourne le framerate courant (entier)
graphics.draw_text(text, x, y, size [,color])Affiche une chaîne à (x,y) avec la taille et la couleur données
graphics.close()Ferme la fenêtre et libère les ressources

Couleur personnalisée via les opérateurs bits :

## couleur aléatoire claire
var r = math.floor(math.rand() * 200) + 55
var g = math.floor(math.rand() * 200) + 55
var b = math.floor(math.rand() * 200) + 55
